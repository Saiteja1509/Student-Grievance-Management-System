# Student Grievance Management System

## Overview
A console-based Java application developed using Object-Oriented Programming principles to manage student grievances.

## Features
- Submit grievances
- View grievances
- Resolve grievances
- Student and Admin roles
- OOP concepts implementation

## Technologies Used
- Java
- OOP
- Collections Framework
- Exception Handling

## Project Structure
- Main.java
- User.java
- Student.java
- Admin.java
- Grievance.java
- GrievanceService.java
- GrievanceServiceImpl.java
